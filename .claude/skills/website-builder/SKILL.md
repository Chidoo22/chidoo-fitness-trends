# Website Builder Skill

## What This Skill Does
Builds complete websites and web pages — from single landing pages to multi-section sites. Produces clean, professional HTML/CSS/JS files that work on Windows without any server setup.

## Invoke When
- "Build a website for..."
- "Create a landing page for..."
- "Make a portfolio page..."
- "Design a business website..."
- "Create a webpage that..."
- "Build a one-page site..."

## Capabilities

### Website Types
- Business / company websites
- Portfolio websites
- Product landing pages
- Service pages
- About / bio pages
- Simple e-commerce layouts (static, no backend)

### Tech Stack
- HTML5 + CSS3 + Vanilla JavaScript (no framework needed to open)
- Optional: Tailwind CSS (via CDN — no install required)
- Responsive design (works on phone and desktop)
- Animations and smooth scrolling
- Contact forms (visual layout — backend connection separate)

### Design Styles Available
- Clean & minimal (white space, simple fonts)
- Bold & modern (dark theme, strong typography)
- Professional corporate (navy/grey palette)
- Warm & friendly (rounded, soft colors)

## Output
- Single `.html` file saved to `projects/websites/`
- Open directly in any browser — no server needed
- Fully editable

## Rules
1. Always ask: "What is the ONE thing this website needs to make the visitor do?"
2. Every website must be mobile-responsive
3. No external dependencies that require installing packages — CDN links only
4. Save all outputs to `projects/websites/[project-name].html`
5. Use reference files in `01-KNOWLEDGE/website-references/` for style guidance
